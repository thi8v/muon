# The Muon Programming Language

> [!WARNING]
> The compiler, `muonc` and the programming language are both experimental, and
> **may not** work. It is not recommended to use Muon in production.

Muon is a general-purpose programming language, that compile AOT to machine code,
used to create maintanable, reusable and optimized software.
