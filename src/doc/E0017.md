# Error `E0017`

The if expression is malformed.

Erroneous example:
```muon
if true { 0 } else if (false) 12 else 13 // ERROR: malformed if expression
```

To fix this error you simply make the else branch of the first if to also be a
block if expression. Example:
```muon
if true { 0 } else if false { 12 } else { 13 }
```
